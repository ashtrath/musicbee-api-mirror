# MusicBee API Mirror

An automated mirror repository for the official [MusicBee API](https://getmusicbee.com/forum/index.php?topic=1972.0). 

This repository tracks the official [`MusicBeeInterface.cs`](https://getmusicbee.com/download/plugins/MusicBeeInterface.cs) and [`MusicBeeInterface.vb`](https://getmusicbee.com/download/plugins/MusicBeeInterface.vb) files and automatically generates dated GitHub Releases whenever structural changes occur.
